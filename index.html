What is Web3.0 (Web3)

Web 3.0 is a new idea proposed as a decentralized Internet based on blockchain. Background to the proposal of Web3.0 (Web3)
Web3.0 (Web3, Web3) is attracting attention as a new Internet concept.


Web 1.0 focused on static content
The 1990s and early 2000s are considered the era of Web 1.0.
Like a company's corporate site or a personal homepage,
Its primary use is to display predefined content.
HTML and CSS are the core, and some include Perl and PHP.
Web applications are now created in programming languages.


Web 2.0 where interactivity was born
In the latter half of the 2000s, people began posting articles on online blogs and posting videos.
User-generated content is on the rise. A typical use case is
SNS (social networking service) represented by Twitter, YouTube, Facebook, Instagram, etc.
It will be common for users to interact with each other and organize information by tagging.
On the other hand, as the platform that accumulates user information becomes bloated and monopolized,

Criticisms about privacy have come to be pointed out.


Web3.0 with distributed information ownership
Web3.0 can be thought of as advocated to improve the excesses of centralized Web2.0.
In Web3.0, network participants are not expected to exist like platform companies,
It is based on interconnected P2P (peer-to-peer) connections.
The characteristic of Web 3.0 is that it advocates a democratic idea in which data is held in a distributed manner.
Blockchain is attracting attention as a technology that realizes the concept of Web 3.0.


Blockchain records data transactions on a ledger distributed among network participants.
A mechanism was introduced to maintain data integrity and prevent tampering using cryptographic technology.
